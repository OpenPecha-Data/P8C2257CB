(3)na | | rnam par grol źes śes pa’o | |

byas pa ste | | sred pa gźan ni yod ma yin | |

las | | (4)de mthag tu rnam par grol | |

che ba’i | | ’od sruṅs gsum gyis saṅs rgyas mdzes | |

(5)bor ba’i | | dka’ thub nags de mdzes ma gyur | |

phyir | | sṅon gyi dam bca’ dran pa’o | |

de nas ’od ma’i tshal yul ni | | de bźin gśegs pas rab tu thob | |blon po rnams ni rjes ’gro
bas | | (6)thos nas rgyas po sṅon du soṅ | |

mig | | ri yi lam nas phyi rol byuṅ | |

phyir | | śiṅ rta (7)las ni sa la babs | |

bźin | | thub pa rgyal po sṅon du soṅ | |

la | | mṅon par rjes gnaṅ (61a1)bsdad pa’o | |

mthoṅ nas | | (2)khyed kyis me ni spaṅs pa’o | |
